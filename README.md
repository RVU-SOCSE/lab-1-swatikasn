

# Bias–Variance Trade-off (LAB 1)

This project demonstrates the bias–variance trade-off using
Linear Regression and Polynomial Regression.

## Dataset
Study Hours vs Marks

## Models
- Linear Regression
- Polynomial Regression (Degree 4)

## Tasks
- Model fitting
- Prediction
- Mean Squared Error calculation
- Bias–Variance analysis

## How to Run
pip install -r requirements.txt
python bias_variance_lab1.py
